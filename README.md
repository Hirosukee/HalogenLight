# 蛍光灯mod
![image](https://user-images.githubusercontent.com/51872161/133731550-f616262d-65c4-4dfd-adbf-2662936a03c3.png)  

某さんのmodがDLできなくなっていたので自分用に作りました。  
壁、床、天井に置ける蛍光灯を追加します。

スレ: https://forum.civa.jp/viewtopic.php?f=3&t=796
